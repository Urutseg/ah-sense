# Release Notes

## Packaging

The addon folder is `AhSense/`. Release archives should package this folder as the top-level addon directory.

## Versioning

The project starts at `0.0.0-dev`. Future releases should update:

- `AhSense/AhSense.toc`
- `CHANGELOG.md`
- Packaged release artifact name

## Client Interface

The `.toc` `Interface` value should be reviewed for each supported Retail patch before release.
